# Learn Update

Capture learnings and update context artifacts (Learn phase).

## Context
- Feature has been implemented
- Outcomes and learnings need to be captured
- Context artifacts need to be updated

## Required Inputs
- Feature name (completed)
- Learnings: What worked, what didn't
- Outcomes: Concrete results
- Evidence: Links to execution results

## Steps

1. Load completed feature intent
2. Load related decisions
3. Update feature intent:
   - Mark as Completed
   - Add learnings section
   - Add limitations section
4. Update decisions:
   - Add Outcomes section
   - Document what worked/didn't
5. Update changelog:
   - What changed
   - Why it changed
   - Links to feature/decisions
6. Optionally create new knowledge artifacts (patterns/anti-patterns)

## Output
Updated context artifacts with learnings and outcomes documented.

## Constraints
- Learning must be explicit (never automatic)
- Must reference concrete outcomes
- Must link to evidence
- Must update changelog
- Must not mutate history silently
