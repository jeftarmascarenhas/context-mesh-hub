# New Project Setup

Initialize a new project with Context Mesh Hub.

## Context
- Repository is empty or newly created
- Developer wants to adopt Context Mesh framework
- No existing context artifacts

## Steps

1. Create `context/` directory structure
2. Create `context/intent/project-intent.md` with project vision
3. Create `context/.context-mesh-framework.md` (framework reference)
4. Create `AGENTS.md` (agent instructions)
5. Initialize `context/evolution/changelog.md`

## Output
Generate the initial Context Mesh structure for a greenfield project.
