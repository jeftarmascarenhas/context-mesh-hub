# Existing Project Bootstrap

Bootstrap Context Mesh Hub in an existing repository (brownfield).

## Context
- Repository has existing codebase
- Developer wants to adopt Context Mesh incrementally
- Need to extract initial context from existing code

## Steps

1. Analyze existing repository structure
2. Create `context/` directory structure
3. Extract initial context (decisions, patterns) from codebase
4. Create `context/intent/project-intent.md` based on analysis
5. Create baseline `context/.context-mesh-framework.md`
6. Create `AGENTS.md`
7. Initialize `context/evolution/changelog.md`

## Output
Generate Context Mesh structure with initial extracted context for brownfield project.
